# Bus Ticketing (Assignment-AI)

This repository contains a starter scaffold for the Bus Ticketing assignment, and a minimal FastAPI backend to get started.

Stack used:
- FastAPI
- SQLite
- SQLalchemy
- Reactjs with vite
- ChromaDB

Quick start (Windows `cmd.exe`):

1. Clone the repository:

```
git clone 
```

2. Setup the environments (backend + RAG):
```
pip install -r requirements.txt
```
